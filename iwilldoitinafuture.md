# Будущие функции Jas Volunteers (Clan Edition)

Эти идеи запланированы на следующие этапы разработки:

### 1. Функционал «Вступить / Выйти»
- Кнопка «Вступить в команду» на странице команды (если пользователь еще не состоит ни в какой).
- Система заявок (нужно ли одобрение координатора).
- Кнопка «Покинуть команду» с подтверждением.

### 2. «Клановые войны» (Соревнования между командами)
- Автоматический подсчёт суммарных часов всех участников команды за неделю.
- Таблица лидеров среди команд по недельным часам.
- Бонусные награды / значки для команды-победителя недели.
- Визуальные эффекты (аура) для топ-3 команд на главной странице.
